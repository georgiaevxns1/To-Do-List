# To-Do-List

A simple project and to-do management web app built with vanilla JavaScript as part of the Odin Project. 



## ***Features***

### **Projects**
- Create new projects  
- Edit project names  
- Delete projects (and associated to-dos)  
- View all projects in a navigation sidebar  

### **To-Dos**
- Add to-dos under existing projects  
- Edit existing to-dos (title, description, date, project)  
- Delete to-dos  
- Expand to-dos to view full details  
- Filter to-dos by project  
- View all to-dos across all projects  

### **Data Persistence**
- Projects and to-dos are saved in **localStorage**  
- Data persists after page refresh  


## ***Technologies Used***

- HTML5  
- CSS3  
- Vanilla JavaScript (ES6 Modules)  
- Classes (Project & ToDo models)  
- Modular Architecture (separation of logic, UI, and initialization)  
- Local Storage API (data persistence)  
- DOM Manipulation  
- Event Listeners & Event Delegation
